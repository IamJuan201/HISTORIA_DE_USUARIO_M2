## Description

This project is a responsive web portfolio developed using HTML, CSS, and basic JavaScript.
The main goal is to improve the previous portfolio by making it adaptable to different screen sizes, optimizing styles, and adding simple interactivity.

The user can navigate through the portfolio, view content correctly on mobile and desktop devices, and interact with basic elements like buttons.
The project applies good CSS practices, media queries, and DOM manipulation with JavaScript.

## Functionalities

### Responsive Design

* Adapt content to mobile, tablet, and desktop screens.
* Use media queries for different screen sizes.
* Improve readability on small devices.

### Style Optimization

* Organized and cleaner CSS code.
* Better use of classes and selectors.
* Improved visual structure.

### Basic Interactivity

* Use JavaScript for simple interactions.
* DOM manipulation.
* Buttons with basic actions.
